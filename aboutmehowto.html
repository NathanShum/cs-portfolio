    How To-About Me

    Vocabulary-
    Tag:<?>, most tags need a / before the element to close them. For example <p>?</p>
    Elements: the letters in the < > brackets


  1. <p> tags are used to write paragraphs in your body.

  2. <h1,2,3,4,5,6> tags are used to change the font size of your words.I used this more than the paragraph tag because they provided more varitey than what a p tag does. I put titles as a bigger size and descriptions as a smaller font.

  3. <table> tags are used to make tables for making a schedule or organizing words.

  4. <ul>, <ol>, and <li> are used to make lists and order items with numbers or bullet points. I used this to list some of my hobbies. I put this in the top of my About Me page as kind of a table of contents and a general guid of what will be on that page of info.

  5. <img>tags are used to insert an image. The images I put on my About Me page represent what most of my favorite hobbies look like.

  6. <br> tags skip a line. I used this to make sure my About Me page doesn't look too cramped and I also used it to divide my hobbies as an organization tool.

  7. width="?" can also be used to change the size of the picture you just inserted. I used this to resize my images to hopefully make them look more proportional.

  8. <href> tags are used to insert a link to a website.

  9. A CSS can be linked to a HTML by using <link rel="?" href="?" type="?" />. CSS are used to change the color of words and background, align text, add borders, etc. I linked my About Me page to a CSS to change colors and add borders to my pictures. I also used CSS on this page to put Halloween colors on it.

  10. <em> tags are used to italicise words. This tag is used to emphasize words or phrases that are important to my page.

  11. <strong> tags bold the words inside the tags. This tag is similar to the italicizing tag in the sense that it is used to put an amphasis to something.

  12. <style> can be used to change colors and borders kinda like CSS.

  13. <div> tags seperate wors and pictures to make changing styles in CSS easier. In my About Me page, I used this to put pictures side by side to organize them more. It makes it so that the description I put under the pictures are organized well.

  14. <a href="?"><li>?</li></a> allows you to include a navbar in your page as long as the page is linked to a navbar CSS. I had the navbar linked to both of my pages to make transfering between the pages easier.

  15. Class="?" can be used to give a code a "nickname" which allows you to also use the classname in CSS for changing colors, adding borders, aligning words, etc.
    id="?" is like a lcass, except it only can be applied once.
    In a CSS, a .? { } is needed to change class tags and #? { } is needed for id tags.

  16. On the CSS, img { border-style: inset;} can be used to make it so that the image looks more into the background.
    Text-align:?; can align your words to a side of the page.
    Bakcground-color:?; and color:?; changes the colors of your HTML.
    Border:?; adds a border and margin:?; separates text and pictures. On the About Me page, I added borders to picutures to distinguish the colors of the background and the colors of the picture to make it easier to see.

  17. The navbar can be made by <a href="(whatever page you want your navbar to take you to)"><li>(what words will show on your navbar)</li></a>.
    To change the way your navbar looks like, you can link the CSS for your navbar to your HTML.
